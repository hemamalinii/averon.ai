=============================================================================
API ENDPOINTS
=============================================================================

POST /api/predict
- Input: { transaction, amount?, merchant_name? }
- Output: { category, confidence, influential_tokens, category_id }

POST /api/explain
- Input: { transaction, amount?, merchant_name? }
- Output: { transaction, category, confidence, influences[] }

GET /api/taxonomy
- Output: { categories[] }

POST /api/taxonomy
- Input: { categories[] }
- Output: { success, taxonomy }

POST /api/feedback
- Input: { transaction_id, corrected_category_id, notes? }
- Output: { success, feedback }

GET /api/metrics
- Output: { macro_f1, accuracy, latency_ms, throughput_per_second, per_class_f1 }
